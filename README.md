# ARC-Image-Segmentation-Task
Image Segmentation Model For Automotive Research Center


## Requirements
- Python 3
- Tensorflow
- Matplotlib
- Tensorflow_hub
- Numpy



Model Name : HRNet <br>
Dataset : CamVid <br>

The CamVid video dataset was split into individual frames. Sample images have been provided in the repository.
